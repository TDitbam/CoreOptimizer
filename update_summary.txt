2. ADVANCED CORE AFFINITY LOGIC
-------------------------------
* Hybrid Architecture Optimization: Accurate detection of P-Cores and E-Cores.
* Core 0 Management: Added explicit setting to "Exclude Core 0" to reserve it for OS background tasks.
* Three Working Modes:
    - P-CORE: High performance on Performance cores (Priority: HIGH).
    - E-CORE: Offload tasks to Efficiency cores (Priority: BELOW_NORMAL).
    - NORMAL: Standard Windows thread management.

-------------------------------------------
* Managed Games: Target specific applications by executable filename.
* Managed Directories: NEW! Designate entire folders (e.g., D:\Games) as auto-optimization zones. Any game launched from these folders is automatically optimized.
* Smart PC Scanner: Built-in scanner to discover installed games in common directories.

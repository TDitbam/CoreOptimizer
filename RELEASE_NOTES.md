# 🚀 บันทึกการอัปเดต (Release Notes) - CorePriority Pro v2.0.0

## 📝 ภาพรวม (Overview)
อัปเดตเวอร์ชัน 2.0.0 มุ่งเน้นไปที่การเพิ่มความเสถียรและรีดประสิทธิภาพสูงสุด โดยการปรับมาตรฐานใหม่ให้ใช้เฉพาะ **P-Cores (Performance Cores)** สำหรับโปรแกรมที่ต้องการพลังประมวลผลสูง เพื่อตัดปัญหาอาการกระตุก (Micro-stuttering) ที่เกิดจาก E-Cores

## 🌟 มีอะไรใหม่ (What's New)
- **มาตรฐาน P-CORE ใหม่**: ยกเลิกโหมด "HIGH" แบบเดิมที่เคยใช้ทุกคอร์ แล้วเปลี่ยนมาใช้โหมด **P-CORE** เป็นมาตรฐานแทน เพื่อบังคับให้เกมรันบนคอร์ประสิทธิภาพสูงเท่านั้น
- **ปรับปรุงการตั้งค่าเริ่มต้น**: ไฟล์ `config.ini` ที่สร้างขึ้นใหม่จะตั้งค่าเกมยอดนิยม (เช่น Black Desert, CS2, Cyberpunk 2077) ให้รันในโหมด P-CORE โดยอัตโนมัติ
- **ปรับปรุงหน้าจอใช้งาน (GUI)**: ปรับเมนูในหน้า Settings ให้เรียบง่ายขึ้น มีให้เลือก 3 โหมดชัดเจน: **P-CORE**, **E-CORE**, และ **NORMAL**
- **ไฟล์ .exe แบบพกพา**: สร้างไฟล์ `CoreOptimizer.exe` แบบไฟล์เดียวจบ (Single File) ไม่ต้องติดตั้งโปรแกรมเสริมอื่นๆ

## 🛠 รายการแก้ไข (Bug Fixes)
- แก้ไขบั๊กการสร้างไฟล์ `config.ini` ที่เดิมอาจทำให้โปรแกรมความสำคัญสูงเผลอไปใช้ E-core
- ปรับปรุงระบบตรวจจับโฟลเดอร์เกม (Path-based matching) ให้ทำงานได้รวดเร็วและแม่นยำขึ้น

## 🎮 วิธีใช้งาน (How to Use)
1. ดาวน์โหลดไฟล์ `CoreOptimizer.exe`
2. **ต้องคลิกขวาแล้วเลือก "Run as Administrator"** (จำเป็นสำหรับการจัดการคอร์ CPU)
3. เพิ่มชื่อไฟล์เกม หรือโฟลเดอร์เกมในหน้า **Settings**
4. กลับไปที่หน้า Dashboard แล้วกด **START OPTIMIZER**

## 📦 ข้อมูลทางเทคนิค (Technical Details)
- **เครื่องมือที่ใช้สร้าง**: PyInstaller 6.20.0
- **อินเตอร์เฟส**: CustomTkinter 5.2.2
- **ระบบปฏิบัติการที่รองรับ**: Windows 10/11 (แนะนำเป็นพิเศษสำหรับ Intel Gen 12 ขึ้นไป)
